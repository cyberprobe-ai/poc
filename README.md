# CyberProbe AI

AI エージェントによる自動ペネトレーションテストシステム

## Getting Started

### Prerequisites

* Python 3.13+
* [mise](https://github.com/jdx/mise)
* [pdm](https://github.com/pdm-project/pdm)

### Installation

```shell
pdm install
```

### Run Application

```shell
# 1. Run as CLI
mise run-cli

# 2. Run as WebUI
mise run-webui
```
